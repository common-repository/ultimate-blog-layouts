WordPress plugin to add blog layouts for Gutenberg editor.

== Description == 

Ultimate Blog Layouts for Gutenberg allows users to display posts easily in a grid or list. This plugin works perfectly with the Gutenberg editor and uses 2 different blocks for Grid and List. 

Each block comes with 3 templates(layouts) with multiple options like category based query, enable or disable meta info (author, date, comments, etc), image size selection, excerpt length, post order and more.

As the plugin is fully compatible with Gutenberg, you can show your posts either in a grid or a list anywhere on a page you like using blocks.

== Installation ==

1. Upload the plugin files to the **`/wp-content/plugins/`** directory, or into admin area of WordPress visit **`Plugins -> Add New`** and search **`Ultimate Blog Layouts`**.
2. Install & Activate the plugin through the **`Plugins' page`** in WordPress.
3. After the plugin is activated you cango to the page and add it.
